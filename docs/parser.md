## EXPRESSION

```
EXPRESSION = (VARAIBLE + REDUNDENT + '=' + REDUNDENT + EXEPRESSION | MULDIV_OPERATIN | ADDSUB_OPERATION | OPERAND)? + REDUNDENT
OPERAND = RETURNABLE_COMMAND | VARAIBLE | VALUE | (EXPRESSION) | NEGATIVE_EXPRESSION

MULDIV_OPERATION = OPERAND + REDUNDENT + ('*' | '/') + REDUNDENT + OPERAND
ADDSUB_OPERATION = OPERAND + REDUNDENT + ('+' | '-') + REDUNDENT + OPERAND
```
