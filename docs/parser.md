## EXPRESSION

```
EXPRESSION = (
  case 'assign':
    VARAIBLE + REDUNDENT + '=' + REDUNDENT + EXEPRESSION
  case 'operation':
    MULDIV_OPERATION | ADDSUB_OPERATION
  case 'operand':
    OPERAND
  )? + REDUNDENT
OPERAND = RETURNABLE_COMMAND | VARAIBLE | VALUE | (EXPRESSION) | NEGATIVE_EXPRESSION
VARIABLE = NAME?
NEGATIVE_EXPRESSION = '!' + EXPRESSION
MULDIV_OPERATION = OPERAND + REDUNDENT + ('*' | '/') + REDUNDENT + OPERAND
ADDSUB_OPERATION = OPERAND + REDUNDENT + ('+' | '-') + REDUNDENT + OPERAND

```
